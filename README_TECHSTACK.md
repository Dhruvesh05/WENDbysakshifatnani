## Tech Stack

- **Language:** TypeScript / TSX
- **Framework:** React
- **Bundler / Dev server:** Vite
- **Styling:** Tailwind CSS (v4) + PostCSS; also uses Emotion and MUI
- **UI primitives:** Radix UI, Material UI, Lucide icons
- **Routing & Forms:** react-router, react-hook-form
- **Build tooling:** @vitejs/plugin-react, @tailwindcss/vite
- **Package manager hint:** `pnpm` overrides present in `package.json` (pnpm recommended)
